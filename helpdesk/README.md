# Helpdesk - Troubleshooting Turma A

Guias de diagnóstico que o Viriato consulta quando há erros na infraestrutura.

Cada arquivo `.md` aqui é um problema conhecido com causa provável e solução passo-a-passo. O Viriato lê estes guias automaticamente e usa o conteúdo para responder ao utilizador quando algo falha.

## Arquivos
- `biblioteca-nao-indexa.md` — upload aceito mas Viriato não vê o documento
- `viriato-sem-resposta.md` — Viriato não responde ou demora muito
- `upload-falha.md` — falha ao anexar documento ou indexar
- `dados-sumiram.md` — eventos/escala/checklist desapareceram
- `parada-pelo-governador.md` — frase humorada padrão para qualquer erro inesperado
